## サイト概要
### サイトテーマ
- 景色の写真を投稿するsnsサイト 写真の場所を登録しgoogle mapで表示させる

### テーマを選んだ理由
- 自分自身が自然の景色を見るのが好きで、写真撮影も好き
- この写真の場所に行きたいと思っても、場所がわかりづらかったりして行きづらいことがあった

### ターゲットユーザ
- 20代~40代
- 旅行や写真好きな人
- 景色の写真を見るのが好きな人

### 主な利用シーン
- 自然の景色を見に行きたいと思ったときに候補探し
- 自然の写真を見て癒されたいとき
- 自分の感動した景色を他の人に共有したいとき
